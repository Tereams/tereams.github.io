CMB Birefringence Simulation & SCNN Modeling (In Progress)

Simulating CMB birefringence maps under axion-string loop-crossing models, and experimenting with CNN/SCNN architectures to see whether we can recover hidden physical parameters directly from polarization rotation patterns.

The project aims to combine cosmology and machine learning in a hands-on way— generating synthetic birefringence maps and testing whether neural networks can decode the underlying string-network properties.

Tech: Python, Jax, PyTorch

Pendulum & Chaos Simulation

A modular C++ implementation for simulating nonlinear systems such as the single and double pendulum. It includes RK4/RK45 integrators and produces clean CSV outputs for visualization and analysis.

Python tools are provided to generate time-series plots, phase portraits, Lyapunov behavior, and Poincaré sections. The project serves as a small sandbox for studying chaos and sensitivity to initial conditions.

Tech: C++17, Python (NumPy, Matplotlib)

Self-Organizing Map for Unsupervised Clustering of Hyperspectral Data

An unsupervised learning project using a Self-Organizing Map (SOM) to cluster high-dimensional AVIRIS hyperspectral signatures. The model projects spectra onto a 2D grid that preserves similarity, making structure easier to interpret.

This project highlights how classical unsupervised methods can reveal meaningful patterns in complex datasets without requiring labels, and provides intuitive visualizations for exploring spectral relationships.

Tech: Python, scikit-learn
